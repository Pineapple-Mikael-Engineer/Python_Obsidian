# Funciones constructoras de tipos

Son funciones built-in usadas para convertir o crear valores de un tipo concreto:

- `int()`
- `float()`
- `str()`
- `list()`
- `tuple()`
- `dict()`
- `set()`

Relacionado: [[../01 Variables y Tipos de Datos/03 Transformación de Tipos|Transformación de tipos]].
